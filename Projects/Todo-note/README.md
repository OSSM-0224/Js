# 📝 Todo Note App

A simple and efficient **Todo Note Application** built to manage daily tasks and notes. This project allows users to create, update, delete, and manage notes easily with a clean backend structure and API integration.

---

## 🚀 Features

✔️ Create New Notes  
✔️ View All Notes  
✔️ Update Existing Notes  
✔️ Delete Notes  
✔️ REST API Integration  
✔️ JSON-based Request Handling  
✔️ Organized Backend Structure

---

## Instructions before using the REPO install express and mongoose

### Backend

- Node.js
- Express.js

### Database

- MongoDB
